# TransportesX

Página web con backend (Express + Handlebars) y frontend (React).

## Estructura
- `backend/` - servidor Node/Express (admin)
- `frontend/` - app React

## Cómo ejecutar en local
1. Instala dependencias en la raíz (usa Concurrently para ejecutar ambas):

```bash
npm install
npm run start
```

2. Backend: `cd backend && npm install && npm run dev`
3. Frontend: `cd frontend && npm install && npm start`

## Ignorar archivos sensibles
Asegúrate de no subir claves ni archivos de entorno al repo. Usa `.gitignore`.
